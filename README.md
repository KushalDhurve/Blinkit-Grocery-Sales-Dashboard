# Blinkit-Grocery-Sales-Dashboard

## Project Overview
Interactive Power BI dashboard analyzing Blinkit grocery sales data.

## Tools Used
- Power BI
- Excel
- DAX

## KPIs
- Total Sales
- Average Sales
- Total Items
- Average Rating

## Dashboard Preview

![Dashboard](https://github.com/KushalDhurve/Blinkit-Grocery-Sales-Dashboard/blob/main/(1)%20Blinkit%20Dashboard%20Overview.png)

## Insights
- Supermarket Type 1 generated highest sales.
- Tier 3 locations contributed maximum revenue.
- Medium-sized outlets performed best.
